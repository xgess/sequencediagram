// Example 21: Linear and Parallel Positioning
// Demonstrates linear vs parallel message layouts

title Linear vs Parallel Demo

participant A
participant B
participant C

// Default mode: messages at different Y positions
A->B:Message 1
B->C:Message 2

space

// Linear mode: messages at same Y level
linear

A->B:Linear 1
B->C:Linear 2

space

linear off

// Parallel mode: independent messages happen simultaneously
parallel

A->B:Parallel to B
A->C:Parallel to C

parallel off

space

// Normal sequential flow again
B-->A:Response from B
C-->A:Response from C
