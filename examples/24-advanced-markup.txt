// Example 24: Advanced Text Markup
// Demonstrates all text markup options

title Advanced Text Markup Demo

participant User
participant System

// Basic markup
User->System:**Bold** message
System-->User://Italic// response

// Combinations
User->System:**//Bold italic//**
System-->User:__Underlined__ text

// Strike-through
User->System:~~Cancelled~~ request

// Color and size
User->System:<color:#red>Error</color> message
System-->User:<color:#green>Success</color>

User->System:<size:14>Large</size> and <size:10>small</size>

// Subscript and superscript
User->System:H<sub>2</sub>O formula
System-->User:E=mc<sup>2</sup>

// Links
User->System:<link:https://example.com>Click here</link>

// Combined formatting
System-->User:**<color:#blue>Bold blue</color>** text

note over User:**Bold** and //italic//\n<color:#red>Colored</color> note
