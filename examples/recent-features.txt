// --- Participant Types ---

// Standard participant with styling
participantstyle #lightyellow #orange;2
participant Alice
participant Bob

// Database with dashed border (BUG-029 fix)
database MyDB #lightblue #darkblue;2;dashed

// Actor with styling
actor User #lightgreen #green

// --- Styled Messages ---

// Named style for warnings
style warning #ffffcc #red;2,<b>

// Colored arrow (BUG-028 fix)
Alice-[#red]->Bob: Red arrow message
Bob-[#blue]->Alice: Blue arrow message

// Using named style on message
Alice-[##warning]->Bob: Warning style!

// Multiline message (positioned above arrow - BUG-019 fix)
Bob->MyDB: Query database\nwith multiline\nlabel

// --- Unknown Participant Error (BUG-032 fix) ---
// Uncomment the line below to see the error indicator
// Alice->UnknownParticipant: This would show error

// --- Notes with Styling ---

notestyle #ffffcc #orange
note over Alice: Default note style applied

note over Bob #lightgreen #darkgreen: Custom styled note

// --- Database Interactions ---

User->MyDB: Connect
activate MyDB #lightblue
MyDB-->User: Connection OK
deactivate MyDB

// --- Fragments ---

alt Success Path
  Alice->Bob: Request
  Bob->Alice: Response OK
else Error Path
  Alice->Bob: Request
  Bob-[#red]->Alice: Error!
end
