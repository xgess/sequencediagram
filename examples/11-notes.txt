// Example 11: Notes and Boxes
// Demonstrates note, box, abox, rbox with positioning

title Notes Demo

participant Client
participant Server
participant Database

// Basic note over a participant
note over Client:User initiates request

Client->Server:Request data

// Note spanning multiple participants
note over Client,Server:Communication established

// Notes to left/right
note left of Client:Client-side\nprocessing
note right of Server:Server handles\nthe request

Server->Database:Query

// Box variations
box over Database:Standard box note
abox over Database:Angular box\n(hexagonal)
rbox over Database:Rounded box

// Styled notes
note over Server #lightblue:Styled note\nwith fill color
box over Client #yellow #red;2:Box with border\nand styling

Database-->Server:Results
Server-->Client:Response

note over Client,Server,Database #lightgreen:Request completed successfully
