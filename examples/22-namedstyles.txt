// Example 22: Named Styles
// Demonstrates defining and using named styles

title Named Styles Demo

// Define named styles
style error #red
style success #green
style warning #orange

participant Client
participant Server
participant Database

Client->Server:Login request

// Use named style with ##styleName syntax
Server->Database:Validate user
Database-[##success]->Server:Valid

Server-[##success]->Client:Login OK

space

Client->Server:Bad request
Server-[##error]->Client:400 Bad Request

space

Client->Server:Slow operation
Server-[##warning]->Client:Processing (may be slow)

Server->Database:Complex query
Database-->(2)Server:Results

Server-->Client:Complete
