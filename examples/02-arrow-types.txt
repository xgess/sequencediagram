// Example 2: Arrow Types
// Demonstrates all four arrow types

title Arrow Types Demo

participant A
participant B

// Synchronous request (solid line, solid arrowhead)
A->B:Sync request

// Synchronous response (dashed line, solid arrowhead)
B-->A:Sync response

// Asynchronous request (solid line, open arrowhead)
A->>B:Async request

// Asynchronous response (dashed line, open arrowhead)
B-->>A:Async response
