// Example 14: Autonumbering
// Demonstrates automatic message numbering

title Autonumber Demo

participant Client
participant Server
participant Database

// Turn on autonumbering (starts at 1)
autonumber 1

Client->Server:Login request
Server->Database:Validate credentials
Database-->Server:Valid
Server-->Client:Login success

// Change the start number
autonumber 10

Client->Server:Get profile
Server->Database:Query user
Database-->Server:User data
Server-->Client:Profile data

// Turn off autonumbering
autonumber off

Client->Server:Background sync
Server-->Client:Sync complete

// Resume with new number
autonumber 100

Client->Server:Logout
Server->Database:Clear session
Database-->Server:Done
Server-->Client:Logged out
